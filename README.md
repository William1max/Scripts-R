# Scripts-R
E ai, blz? Neste repositório estou com partilhando scripts para manipulação de dados e criação de gráficos que foi desenvolvido para uso no Laboratório de pesquisa (LAMMA) da faculdade de Meteorologia (ICAT) da Universidade Federal de Alagoas (UFAL)   


Estou compartilhando um Script para a criação de dendograma 
  Dendrograma é um tipo de visualização gráfica usada para representar a estrutura de agrupamento em dados hierárquicos.
  É frequentemente usado em análises de cluster para mostrar como os objetos são agrupados em clusters, com base em suas similaridades ou distâncias.
  O dendrograma permite identificar padrões nos dados e pode ajudar a selecionar o número ideal de clusters ou a avaliar a qualidade dos clusters formados. 
  a principio vai ser necessario instalar alguns pacotes (install.packages('nome_do_pacote'))que é exibido nos scripts.
